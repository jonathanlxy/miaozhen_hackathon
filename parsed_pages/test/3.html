宝马X2量产版曝光：老X1换代重生？
2017-10-11 09:41:02  作者： 鲲鹏 编辑：鲲鹏 评论 ( 0 )
让小伙伴们也看看：
收藏文章

去年，宝马首发了旗下的全新X2概念车，主打跨界风格，视觉效果非常不错。现在，外媒拍摄到了新车的量产版低伪装谍照。

整体来说，新车基本上保留了X2概念车的设计，只是细节方面为量产做了优化。 前脸造型颇为扁平，大灯的造型十分修长。 但这次曝光的应该是普通版本，并没有加入M运动套件。

从侧面来看，C柱上保留了概念车上的宝马品牌LOGO。 而在车门下方的黑色塑料板上，采用了三个六边形的装饰槽，凸显了跨界风格。

由于全新X1放弃了跨界造型，转而采用传统的造型， 未来X2上市之后会成为宝马旗下唯一一款跨界SUV产品。 因此，有网友认为X2其实才是真正的换代X1。

此前消息称，宝马X2同宝马X1一样基于UKL平台打造，将提供1.5T/2.0T汽油发动机可选，2.0T高功率版版的最大功率为231马力，峰值扭矩350牛·米，传动系统匹配8速手自一体变速箱。
